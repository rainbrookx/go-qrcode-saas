# 二维码SaaS平台

## 项目定位

本系统是开源二维码SaaS平台，因此不设置付费功能

## 主要功能

| Tab 栏名称 | 简介                                                                                                                   | 是否登录 |
| ---------- | ---------------------------------------------------------------------------------------------------------------------- | -------- |
| 文本       | 普通文本生成二维码                                                                                                     | 否       |
| 网址静态码 | 和“普通文本生成二维码”本质上一样，但是需要单独列出，并添加 placeholder “http(s)://”                                    | 否       |
| 网址跳转码 | 后端生成短链接，用户扫描后跳转到设置的网址                                                                             | 是       |
| 文章       | 后端生成文章专属短链接，用户扫描后跳转到设置的文章（富文本：H5、图片、视频、音频、文件附件），要限制附件大小，文章字数 | 是       |
| 表单       | 后端生成表单专属短链接， 拖拽生成表单，JSON存储表单配置                                                                | 是       |
| 解码       | 解析二维码                                                                                                             | 否       |

### 表格字段说明

- **是否登录**：如果为“否”，则前端生成二维码，如果为“是”，则后端生成二维码

## 对“主要功能”的详细描述

### 前端 UI 组件库

前端统一使用 **Ant Design**。

### 功能英文代号

| 业务 | 英文代号 |
| ---- | ---- |
| 文本 | `text` |
| 网址静态码 | `url` |
| 网址跳转码 | `urldyn` |
| 文章 | `article` |
| 表单 | `form` |
| 解码 | `deqr` |

### 后台

当前 MVP 阶段，**不设置 admin 后台**。

---

### 1. 网址跳转码 / 文章 / 表单 的短链接归属

- 短链接默认有效期 **30 天**，用户可自定义有效期，**最长 60 天**。
- 登录后允许修改目标 URL / 文章内容 / 表单配置。
- 需要记录访问统计。

### 2. 文章功能限制

- 富文本编辑器：**TipTap**。
- 正文上限 **50,000 字**。
- 附件类型与单文件大小上限：
  - 图片：5 MB
  - 视频：50 MB
  - 音频：20 MB
  - 通用文件：50 MB
- **视频与通用文件附件之和最多为 1**：用户上传视频或通用文件附件二选一，且最多一个，也可以都不上传。
- 禁止上传可执行文件（`.exe` / `.bat`）和脚本文件（`.js` / `.php` / `.py`）。
- 附件存储：**MinIO**。

### 3. 表单功能

- 基于 Ant Design，支持字段类型：单行文本、多行文本、单选、多选、下拉、日期、文件上传、评分。
- 支持导出 **CSV / Excel**。
- **只支持匿名提交**，提交者无需登录。
- 创建者需配置以下任一约束（也可全部不限）：
  - 提交数量上限（可设为无限制）
  - 截止时间（可不设置）

### 4. 解码功能

- 仅支持**上传图片**进行解码。
- 文件大小上限 **5 MB**。

### 5. 二维码样式定制（所有 Tab 共用）

- **不支持**二维码样式定制。
- 风格为标准二维码样式。
- 输出格式：**PNG**。

### 6. 用户体系

- 注册方式：**邮箱 + 密码**。
- 支持邮箱验证登录、找回密码。
- 用户配额：**100 个活码**（活码 = 网址跳转码 / 文章 / 表单）。

### 7. 短链接规范

格式：

```
https://<主站域名>/<前缀>/<短码>
```

- 字符集：**Base62**
- 长度：**固定 8 位**
- **全小写**
- **不可预测**：需对短码打乱，防止被遍历爬取
- 安全：需防 XSS、防跳转劫持、防恶意内容

**绝对禁止使用的字符**：

- 易混淆字符：`0` `O` `o` `1` `I` `l`（零 / 欧 / 壹 / 爱 / 小写 L）
- 特殊符号：`_` `-` `~` `!` `@` `#` `$` `%` `^` `&` `*` `(` `)`
- 空格、中文、emoji、反斜杠

短链接的过期时间与对应二维码设置的过期时间一致。

#### 业务路由映射表

| 业务标识 | 英文代号 | 路由前缀 | 短链接格式示例 |
| ---- | ---- | ---- | ---- |
| 网址跳转码 | `urldyn` | `u` | `xxx.com/u/<8位短码>` |
| 文章 | `article` | `a` | `xxx.com/a/<8位短码>` |
| 表单 | `form` | `f` | `xxx.com/f/<8位短码>` |

### 8. 未登录生成的二维码（文本 / 网址静态码 / 解码）

- **纯一次性，不落库**。
- 即使用户已登录，也**不支持保存**这三类二维码（文本 / 网址静态码 / 解码）。
